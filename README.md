# GeoCoolAI

## Optimizing Urban Heat Mitigation and Cooling Strategies using AI/ML

### Project Overview
GeoCoolAI is an AI-powered geospatial decision support system that identifies urban heat hotspots, analyzes the factors contributing to urban heating, and recommends optimized cooling strategies such as urban greening, cool roofs, and water bodies.

### Problem Statement
Develop a geospatial AI/ML-based system to:
- Identify urban heat stress hotspots
- Analyze the drivers of urban heating
- Model urban heat dynamics using AI/ML
- Simulate cooling interventions
- Recommend the optimal mitigation strategy
